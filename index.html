🔑 One-time setup: add your Google Maps API key

This tool uses Google's satellite imagery (free for up to 10,000 map views/month). To activate it:

  1. Go to console.cloud.google.com/google/maps-apis and create a project (e.g. “SHIELD Solar”). Add a billing card when prompted — you won't be charged within the free tier.
  2. Enable two APIs: Maps JavaScript API and Places API (New).
  3. Go to Keys & Credentials → Create credentials → API key. Copy it.
  4. Open this HTML file in a text editor, find the line near the top of the script:
    const GMAPS_KEY = "AIzaSyCKAXUyX8ZnviGhDmngRAlnLR1oH3HbM6k";
    and paste your key between the quotes. Save, reload.
  5. Once the tool is hosted online, edit the key in the console and restrict it to your websites (e.g. shieldproject.co.uk/* and your *.netlify.app/* address). Note: a referrer-restricted key won't work when opening the file directly from your computer — restrict only after moving to hosting, or keep a separate unrestricted key for local use.
  6. Recommended: in the console under Quotas, cap Maps JavaScript API at ~500 map loads/day so billing can never surprise you.
